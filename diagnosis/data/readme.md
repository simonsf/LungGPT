Due to the sensitivity of clinical data, it will not be made public. 
Desensitized cases are provided for reference instead.

### There are two JSON files here, namely train_example.json and test_example.json.

##### "input": 

represents the input of the model,
The input must include the patient's chief complaint, present medical history, past medical history. 
Other information beneficial for doctors' diagnosis, such as physical examination findings, 
can also be included in the input simultaneously.

##### "output":

represents the doctor's diagnosis.
